# Point of Sale

a simple pos accepting customer payments.

- Admin creates product
- User Login and Logout
- User input orders

### Inventory

- product code
- product price
- product available qty product  

### Sales
- quantity 
- total sales

### Payment
- cash
- credit

### Receipt
- Orders
- total
- mode of payment
- change
- cashier name
